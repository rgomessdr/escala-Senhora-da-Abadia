# Pasta para Arquivos Estáticos

Para usar sua logo personalizada no sistema:

1. Arraste o seu arquivo de imagem para esta pasta (`public`).
2. Renomeie o arquivo para **`logo.png`** (em letras minúsculas).
3. O sistema reconhecerá automaticamente e substituirá o ícone padrão pela sua imagem.

O arquivo deve ser preferencialmente um PNG com fundo transparente.
