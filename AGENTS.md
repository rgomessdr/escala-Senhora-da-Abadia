# Project Context
This is a management system for Altar Servers (Acólitos & Coroinhas) for the Abadia Sidrolândia Parish in MS, Brazil.

## Business Rules
- Acólitos and Coroinhas have specific assignment rules based on day of the week, week of the month, and location.
- Automatic scheduling must respect these constraints.
- Deployment is intended for Vercel/GitHub.

## Tech Stack
- React + Vite
- Tailwind CSS
- Supabase (Auth + Database + Realtime)
- motion (for animations)
